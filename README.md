# 基于光输出的杨氏双缝干涉实验仿真系统

## 项目简介

本项目基于物理光学中杨氏双缝干涉理论，构建了一个以**光输出（光强分布）**为核心研究对象的数值仿真系统。项目采用 MATLAB 平台进行建模与可视化，实现了单色光与白光条件下双缝干涉现象的仿真研究，可用于物理光学课程教学、实验预演以及相关课程论文研究。

通过对波长、双缝几何参数以及传播距离等物理量的参数化控制，系统能够直观展示干涉条纹随参数变化的规律，为理解光的波动性及干涉机制提供有效辅助。

---

## 理论基础

在夫琅禾费近似条件下，双缝干涉图样可视为单缝衍射包络与双缝干涉条纹的叠加。屏幕上某一点的光强分布可表示为：

[
I(x) = I_0 \left( \frac{\sin \beta}{\beta} \right)^2 \cos^2 \delta
]

其中：

[
\beta = \frac{\pi a x}{\lambda L}, \quad
\delta = \frac{\pi d x}{\lambda L}
]

对白光干涉情形，采用多波长光强叠加的方法，对可见光波段内的干涉结果进行数值求和并归一化处理，从而模拟实际白光干涉实验现象。

---

## 功能特性

* 支持 **单色光 / 白光** 两种干涉模式切换
* 基于光输出的 **干涉光强分布仿真**
* 参数手动输入，支持精确控制：

  * 光源波长或波长范围
  * 双缝间距
  * 单缝宽度
  * 双缝至屏幕距离
* 干涉条纹的 **实时可视化绘制**
* 采用函数主文件结构，代码清晰、稳定性高

* MATLAB R2018a 及以上版本（推荐）

无需额外工具箱。

## 使用方法

1. 克隆或下载本仓库至本地：

   ```bash
   git clone https://github.com/zhouxing111/-matlab-/.git
   ```

2. 将 MATLAB 当前路径切换至项目目录

3. 在 MATLAB 命令行中运行：

   ```matlab
   Young_GUI
   ```

4. 在图形界面中：

   * 选择干涉模式（单色光或白光）
   * 手动输入实验参数
   * 点击“生成干涉图像”即可查看仿真结果

## 仿真说明

* 所有光强结果均进行了归一化处理，便于不同参数条件下的对比分析
* 仿真基于理想条件，未引入环境噪声和器件误差
* 结果适用于理论分析与教学演示，不直接替代精密实验测量
